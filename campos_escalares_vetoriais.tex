\chapter{Campos escalares e campos vetoriais}
 Campo é palavra que se usa para designar funções definidas em uma porção do espaço tridimensional, isto é, funções cujo domínio $D$ é um subconjunto de $\mathbb{R}^3$. Trabalharemos com dois tipos de campos: os campos escalares e os campos vetoriais. Os campos vetoriais são funções cuja imagem é composta de vetores no $\mathbb{R}^3$, já a imagem dos campos escalares são números reais, isto é, escalares.
 
\begin{ex}\label{excampos} São exemplos de campos escalares.
\begin{itemize}
\item [a)] A função que liga a posição de um ponto dentro de uma sala à temperatura neste ponto.
\item [b)] A pressão do ar como função da posição na atmosfera.  
\item [c)] $f(x,y,z)= 100 + 20e^{-\sqrt{x^2+y^2+z^2}}$.
\item [d)] $f(x,y,z)= \vct{r} \cdot \vct{i}$.
\end{itemize}
\end{ex}   


\begin{ex}\label{excampos} São exemplos de campos vetoriais.
\begin{itemize}
\item [a)] A função que liga a posição de um ponto dentro de uma fluido à velocidade (vetor) neste ponto.
\item [b)] O campo magnéticos, elétrico, gravitacional etc.  
\item [c)] $\vct{F}(x,y,z)= x\vct{i} + z\vct{j} - y\vct{k}$.
\item [d)] $\vct{F}(x,y,z)= \vct{r}\times \vct{k}$ .
\end{itemize}
\end{ex}

\section{Campos radiais}
Campos radiais vetoriais \index{campos radiais} são campos da forma $\vct{F}=f(r) \hat{r}$, isto é campos vetoriais cujo módulo depende apenas da distância até a origem, isto é, de $r=\|\vct{r}\|=\sqrt{x^2+y^2+z^2}$ e cuja direção é sempre paralela ao vetor posição, $\vct{r}$.
